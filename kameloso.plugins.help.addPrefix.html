addPrefix

kameloso plugins help

Adds a prefix to a command word; the command prefix if the passed policy is kameloso.plugins.PrefixPolicy.prefixed, the bot nickname if it is kameloso.plugins.PrefixPolicy.nickname, and as is if it is kameloso.plugins.PrefixPolicy.direct.
