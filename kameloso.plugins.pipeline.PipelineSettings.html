Variables

enabled
bool enabled;

Whether or not the Pipeline plugin should do anything at all.

fifoInWorkingDir
bool fifoInWorkingDir;

Whether or not to place the FIFO in the working directory. If false, it will be saved in /tmp or wherever $TMPDIR points. If macOS, then there only if PipelineSettings.useTMPDIR is true.

path
string path;

Custom path to use as FIFO filename, specified with --set pipeline.path=[...].

useTMPDIR
bool useTMPDIR;

Whether ot not to obey $TMPDIR on macOS. If false, the FIFO will be saved in /tmp regardless of $TMPDIR. If true, $TMPDIR will be obeyed.
