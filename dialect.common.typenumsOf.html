typenumsOf

dialect common

Returns the typenums mapping for a given IRCServer.Daemon.

pure @safe nothrow @nogc
auto
typenumsOf
(
const IRCServer.Daemon daemon
)

Parameters

daemon IRCServer.Daemon

The IRCServer.Daemon to get the typenums for.

Return Value

Type: auto

A typenums array of IRCEvents mapped to numerics.

Examples

IRCParser parser;
IRCServer.Daemon daemon = IRCServer.Daemon.unreal;
string daemonstring = "unreal";

parser.typenums = getTypenums(daemon);
parser.client.daemon = daemon;
parser.client.daemonstring = daemonstring;
