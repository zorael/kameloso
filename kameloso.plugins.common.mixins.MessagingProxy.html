MessagingProxy

kameloso plugins common mixins

Mixin to give shorthands to the functions in kameloso.messaging, for easier use when in a with (plugin) { /* ... */ } scope.

This merely makes it possible to use commands like raw("PING :irc.freenode.net") without having to import kameloso.messaging and pass the plugin's IRCPluginState in every call of the functions.
