onAccountInfo

kameloso plugins automode

Potentially applies an automode, depending on the definitions and the user triggering the function.

Different IRCEvent.Types have to be handled differently, as the triggering user may be either the sender or the target.

Additionally none of these events carry a channel, so we'll have to make manual checks to see if the user is in a home channel we're in. Otherwise there's nothing for the bot to do.
