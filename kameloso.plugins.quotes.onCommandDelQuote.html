onCommandDelQuote

kameloso plugins quotes

Removes a quote from the quote database.

On Twitch, selects a quote from the stored quotes of the current channel owner.

The quote is read from the in-memory JSON storage, and it is sent to the channel the triggering event occurred in, alternatively in a private message if the request was sent in one such.
