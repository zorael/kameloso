assertMultilineEquals
void assertMultilineEquals(const(char[]) actual, const(char[]) expected, string file, uint line)

Asserts that two multiple-line strings are equal, with a more detailed error message than the yes/no of the built-in assert().

Examples

enum actual =
"abc
def
ghi
jkl
mno
pqr";

enum expected =
"abc
def
ghi
jkl
mnO
pqr";

actual.assertMultilineEquals(expected);

/*
core.exception.AssertError@some/file.d(123):
Line mismatch at some/file.d:147, block 5:3; expected 'O'(79) was 'o'(111)
expected:"mnO"
  actual:"mno"
            ^
 */
