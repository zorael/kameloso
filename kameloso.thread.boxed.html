Constructor function to create a shared Boxed with an unqualified template type.

shared(Sendable)
boxed
(
T

T payload

Parameters

payload T

Payload whose type to instantiate the Boxed with, and then assign to its internal payload.

Return Value

Type: shared(Sendable)

A shared Boxed!T where T` is the unqualified type of the payload.

Examples

IRCEvent event;  // ...
plugin.state.messages ~= ThreadMessage.busMessage("header", boxed(event));
plugin.state.messages ~= ThreadMessage.busMessage("other header", boxed("text payload"));
plugin.state.messages ~= ThreadMessage.busMessage("ladida", boxed(42));
{
