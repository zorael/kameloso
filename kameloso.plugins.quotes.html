kameloso.plugins.quotes

kameloso plugins

The Quotes plugin allows for saving and replaying user quotes.

A user quote can be added by triggering the "quote" bot command, by use of "!quote [nickname] [quote text...]" (assuming a prefix of "!"). A random one can then be replayed by use of the "!quote [nickname]" command.
