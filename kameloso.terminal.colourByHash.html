colourByHash

kameloso terminal

Shorthand function to colour a passed word by the hash of it.

version(Colours && Colours)
@safe pure nothrow
string
colourByHash
(
const string word
,
const Flag!"brightTerminal" bright
)

Parameters

word string

String to colour.

bright Flag!"brightTerminal"

Whether or not the colour should be adapted for a bright terminal background.

Return Value

Type: string

word, now in colour based on the hash of its contents.

Examples

1 import std.conv : text;
2 
3 {
4     immutable coloured = "kameloso".colourByHash(No.brightTerminal);
